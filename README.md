# Movie app
Movie app with reactJS and OMDB API

## Starting movie app

set the API Key

```js
// windows
>>> SET REACT_APP_API_KEY=YOUR_OMDB_API_KEY
// linux
>>> export REACT_APP_API_KEY=YOUR_OMDB_API_KEY

>>> npm start
```

> You can also replace the API Key in config.js file located in src folder
